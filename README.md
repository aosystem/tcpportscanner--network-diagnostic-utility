# TCP Port Scanner: Network Diagnostic Utility 🔍🌐

**TCP Port Scanner** is a professional-grade network utility built with **Flutter**, designed for IT professionals, network engineers, and security analysts. It provides a lightweight yet powerful way to perform port diagnostics and verify active network services directly from a mobile device.

## 🌟 Key Features
- **High-Speed Detection:** Efficiently scans custom port ranges for open TCP services on any reachable host.
- **Root-Free Operation:** Performs comprehensive scanning without requiring elevated system privileges.
- **Developer-Friendly Output:** Results are presented in a single, selectable text block, making it easy to copy data for logs or external reports.
- **Connectivity Validation:** Ideal for verifying firewall rules, service availability, and IoT device connectivity.
- **Mobile Optimized:** A clean, responsive interface designed for rapid diagnostics in the field.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Utilizes raw TCP socket connections with optimized timeout handling for high-concurrency port probing.
- **Global Support:** Fully localized in **50+ languages**.

*Note: This repository contains the core logic and source code. Assets (images, fonts, and localization files) are excluded to protect intellectual property.*
